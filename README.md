## 📌 About This Project

Task Management System is a web-based application built using Laravel.  
The application is designed to help users manage daily tasks efficiently with a clean and responsive interface.

This project demonstrates implementation of:

- MVC Architecture
- Routing & Controllers
- CRUD Operations
- Database Migration
- Form Validation
- Bootstrap UI Integration

---
## 🚀 Features

- Create new tasks
- Update tasks
- Delete tasks
- Manage task status
- Responsive layout (Mobile & Desktop)
- Clean user interface

---
## 🛠 Tech Stack

- **Backend:** Laravel
- **Frontend:** Bootstrap 5 & Bootstrap Icons
- **Database:** MySQL / MariaDB
- **Local Environment:** Laragon
- **Version Control:** Git & GitHub
